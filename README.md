# todo-list

**Клонируем репозиторий c помощью команды**

```
 $ git clone https://github.com/Studio-Yandex-Practicum/sl_max_frontend.git
```

**Устанавливаем зависимости**

```
$ npm i
```

**Запуск сервера в режиме разработки**

```
$ npm run dev
```

**Создание билда**

```
$ npm build
```

**Деплой проекта**

```
$ npm run deploy
```
